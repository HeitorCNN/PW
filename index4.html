Programação web

Quer aprender a criar sites completos e funcionais, do visual ao codigo que faz tudo acontecer por tras das paginas ? O Curso de programação eb com HTML, CSS e PHP foi desenvolvido para quem deseja dominar as tecnologias fundamentais da web e começar a construir projetos reais, mesmo sem experiencia anterior

Com uma abordagem pratica e acessivel, o curso ensina passo a passo desde a estrutura basica de um site ate a criação de sistemas dinamicos com PHP e banco de dados

Formato e duração

O curso é 100% online, com aulas ao vivo e gravadas, para voce aprender no seu ritmo e de qualquer. As aulas ao vivo acontecem as segundas e quartas, das 19h as 21h, e toda as gravações ficam disponiveis na plataforma para assistir quando quiser, com duração total do curso é de 3 meses.
